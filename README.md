## 🚀 نظرة عامة

**ComptaPro** هو نظام شامل لإدارة الفواتير مطور بـ Flask مع واجهة مستخدم حديثة وتفاعلية. يوفر النظام جميع الأدوات اللازمة لإدارة العملاء والمنتجات والفواتير بطريقة احترافية.

## ✨ المميزات الرئيسية

### 🎯 **الأزرار تعمل بشكل مثالي!**
- ✅ جميع الأزرار تعمل بشكل صحيح
- ✅ معالجة الأحداث التفاعلية
- ✅ التحقق من صحة البيانات في الوقت الفعلي
- ✅ رسائل تنبيه فورية للمستخدم

### 📊 **لوحة التحكم**
- إحصائيات شاملة للنظام
- عدد العملاء والمنتجات والفواتير
- أزرار سريعة للعمليات الأساسية

### 👥 **إدارة العملاء**
- إضافة وتعديل وحذف العملاء
- التحقق من صحة رقم ICE (15 رقم)
- منع التكرار في البيانات
- **جديد**: إضافة عميل مباشرة من نموذج الفاتورة

### 🧾 **نظام الفواتير**
- إنشاء فواتير يدوية متعددة العناصر
- توليد فواتير تلقائية (4000-5000 درهم)
- حساب الضرائب تلقائياً (20% ضريبة القيمة المضافة)
- طباعة ومعاينة الفواتير

### 🎨 **واجهة مستخدم عصرية**
- تصميم متجاوب لجميع الأجهزة
- دعم اللغة العربية مع اتجاه RTL
- ألوان متدرجة وتأثيرات بصرية
- خط Cairo الأنيق

## 🛠️ التقنيات المستخدمة

- **Backend**: Python Flask 2.3.3
- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **UI Framework**: Bootstrap 5.3.0
- **Icons**: Font Awesome 6.4.0
- **Fonts**: Google Fonts (Cairo)
- **Database**: JSON (ملف محلي مع نسخ احتياطية)

## 📦 التثبيت والتشغيل

### المتطلبات
- Python 3.7 أو أحدث
- pip (مدير حزم Python)

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/jalta456/share.git
cd share

# 2. تثبيت المتطلبات
pip install -r requirements.txt

# 3. تشغيل التطبيق
python app.py
```

### الوصول للتطبيق
افتح المتصفح على: **http://localhost:5000**

## 🔧 واجهة برمجة التطبيقات (API)

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| `GET` | `/` | الصفحة الرئيسية |
| `GET` | `/api/data` | جلب جميع البيانات |
| `POST` | `/api/customers` | إضافة عميل جديد |
| `PUT` | `/api/customers/<id>` | تحديث بيانات عميل |
| `DELETE` | `/api/customers/<id>` | حذف عميل |
| `POST` | `/api/invoices` | إنشاء فاتورة جديدة |

## 🎯 كيفية الاستخدام

### 1. إضافة عميل جديد
- انتقل لتبويب "العملاء"
- اضغط "إضافة عميل جديد"
- املأ البيانات المطلوبة
- اضغط "حفظ العميل"

### 2. إنشاء فاتورة
- اضغط "إنشاء فاتورة جديدة" من لوحة التحكم
- اختر العميل أو أضف عميل جديد
- أضف عناصر الفاتورة
- سيتم حساب الضرائب تلقائياً
- اضغط "إنشاء الفاتورة"

### 3. فاتورة تلقائية
- اضغط "إنشاء فاتورة تلقائية"
- سيتم إنشاء فاتورة بقيمة عشوائية (4000-5000 درهم)
